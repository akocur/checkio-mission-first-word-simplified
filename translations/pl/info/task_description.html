Twoim zadaniem jest znalezienie pierwszego słowa w danym ciągu znaków (stringu).

Ta misja to uproszczona wersja misji First Word, którą możesz rozwiązać później.

  • String składa się wyłącznie z liter angielskiego alfabetu i spacji.
  • Spacje nie występują ani na początku, ani na końcu ciągu znaków.

Wejście: String.

Wyjście: String.

Przykład:

{% if interpreter.slug == "js-node" %} 
firstWord("Hello world") == "Hello"
{% else %} 
first_word("Hello world") == "Hello"
{% endif %}

Do czego jest używane: W wierszu poleceń pierwsze słowo jest komendą.

Warunek wstępny: Tekst zawiera litery a-z, A-Z oraz spacje.
